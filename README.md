# 🎨 CSS Practice Projects

A collection of practical UI and animation projects built using **HTML and CSS** to strengthen core frontend fundamentals.

---

## 🚀 About This Repository

This repository contains multiple hands-on projects focused on:

* Layout design
* CSS animations
* Real-world UI replication
* Navbar Designing

---

## 📂 Projects Included

### 🔵 Bouncing Ball Animation

A smooth CSS animation demonstrating motion, timing, and keyframes.

### 🧩 Card Design

Modern UI card component with structured layout and styling.

### 🧱 Designing the Layout

Practice of page layout using Flexbox/Grid for proper alignment and spacing.

### 🌈 Multicolour Navbar

A visually appealing navigation bar with multiple colors and styling techniques.

### 🖥️ UltraEdit Navbar and Body

A replica of the UltraEdit website layout focusing on navbar and body structure.

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox, Grid, Animations)
